# SPEC-TaskGeneration.md

## Purpose

Define how SWE tasks are generated from SPEC files.

## Task Structure

Each task shall contain:

- Task ID
- Title
- Source requirement IDs
- Goal
- Background
- Scope
- Non-scope
- Technical approach
- Acceptance criteria
- Unit tests
- Integration tests
- Definition of Done

## Batch Size

A SWE task should be completable by one developer or agent in 0.5 to 2 working days.

## Task Types

- Schema task
- API task
- Service task
- UI task
- Test task
- Documentation task
- Migration task
- Validation task
