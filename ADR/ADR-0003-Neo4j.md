# ADR-0003 — Neo4j for Regulatory Knowledge Graph

## Status

Proposed

## Context

Regulatory traceability and impact analysis are relationship-heavy problems.

## Decision

Neo4j shall be used as the reference graph database.

## Consequences

Positive:

- Efficient traversal
- Natural impact analysis
- Strong visualization potential

Negative:

- Requires synchronization with primary object store
- Requires graph governance
